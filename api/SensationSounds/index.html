SensationSounds

A set of named UI sound effects for Studio plugins, modeled after Studio Elttob's Sensation library. The module exports a factory: pass it a table of sound asset ids keyed by purpose and it returns the same table after preloading every entry off the main thread. 

local SensationSounds = require(path.to.SensationSounds)
local sounds = SensationSounds({
	generic = "rbxassetid://1234",
	success = "rbxassetid://5678",
})
-- sounds.generic is "rbxassetid://1234"
-- sounds.attention is nil — slot wasn't provided
