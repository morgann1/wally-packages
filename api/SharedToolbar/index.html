SharedToolbar

Cross-plugin shared toolbar built on top of Interposer and [Leader]. Every cooperating plugin calls SharedToolbar.connect with the same toolbarId; the leader builds a single PluginToolbar that mirrors every peer's buttons, and click/state events are forwarded back to the originating peer.

Slight deviation from upstream: drops Fusion entirely. State lives in plain values that get re-applied to the underlying OwnedToolbar when they change, and lifetime is tracked through { unknown } scopes cleaned up via [DoCleanup].
