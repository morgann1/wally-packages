filterMap​

</>
List.filterMap() → ()

Create a copy of a list doing a combination filter and map.

If callback returns nil for any item, it is considered filtered from the list. Any other value is considered the result of the 'map' operation.

sort​

</>
List.sort() → ()

Returns a new list, ordered with the given sort callback. If no callback is given, the default table.sort will be used.
