zzlib

Pure-Lua zlib/gzip/zip decompression, ported from zzlib. Inputs are byte strings; outputs are decompressed byte strings.

zzlib.files(
buf: string--

The full ZIP archive bytes.

) → ()

Returns a stateless iterator over every entry in a ZIP archive.

Each iteration yields (name, offset, size, packed, crc) for one file header. Use zzlib.unzip to extract a specific entry.

unzip​

</>
zzlib.unzip(
buf: string,--

The full ZIP archive bytes.

arg1: string | number,--

Filename to extract, or raw offset.

arg2: number?--

Expected CRC when arg1 is a numeric offset.

) → string--

Decompressed file contents.

Extracts a single file from a ZIP archive.

When arg1 is a string, it's treated as the filename to extract; the function searches the archive and returns the decompressed bytes, or errors if the file isn't present. When arg1 is a number, it's treated as a raw offset into buf and arg2 is the expected CRC.
