Interposer

Cross-plugin RPC layer for Roblox Studio. A BindableEvent parented to StudioService acts as the public medium that every cooperating plugin shares. Each plugin calls Interposer.logon to join the medium, discover other peers, and register typed protocols.

Slight deviation from the upstream: the dev-env name (used by IsLocalDev and the ElttobLog logger prefix) is a parameter rather than the hardcoded "Suite", so plugins can stand up their own Interposer instances without colliding on log output.
